Break the silence under your posts with this simple comments randomizer. 

== Description ==

WPA Comments is a super simple tool for adding many comments with just one click. What it does is taking your preprepared list of comments and user nicknames, and randomizes them to create 'active page' feel to your visitors.

== Installation ==

1. Upload `wpa-comments` directory to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Add your comments/comments authors in proper section (WPA Comments -> Comments / WPA Comments -> Authors). Visit WPA Comments page and add comments.

== Frequently Asked Questions ==

= Saving options not change anything on the frontend page =

This problem might occur if your browser cached page. Try to clean up cache from your browser, or try another browser.

= Cleaning browser's cache didn't helped, what's next? =

Check if you have some caching plugins installed. Try to clean up your site's cache.
